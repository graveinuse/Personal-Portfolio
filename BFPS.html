Overview

BFPS leverages machine learning and multi-source public data (U.S. Census, BLS, Yelp, NOAA) to help entrepreneurs make data-driven decisions. It outputs a composite Feasibility Index (0–1) by predicting 2-year survival probability, ROI, payback period, and seasonality.

The Problem

Many small businesses fail due to insufficient market feasibility assessment. BFPS addresses this gap with a measurable feasibility score built on authoritative datasets, bypassing gut intuition.

The Solution

  • Success Probability (2-year survival) via XGBoost.
  • ROI Forecast via Gradient Boosting.
  • Payback Period via Random Forest.
  • Seasonality Timing via Prophet + NOAA climate features.

Architecture & Models

A robust 5-Layer architecture pipelines user inputs through data integration, ML modeling, feasibility scoring, and UX generation (Dashboard, CLI, API).

Scoring Engine

Feasibility Index =
  0.35 × ROI Prediction
+ 0.25 × Success Probability
+ 0.20 × Payback Period
+ 0.20 × Seasonality

Machine Learning Results

  • XGBoost: 89% Accuracy, 87% Precision.
  • Gradient Boosting (ROI): R² of 0.986, MAE 1.89%.
  • Random Forest (Payback): R² of 0.981.
  • Prophet: Signals extracted from NOAA.

Tech Stack

Data & ML: Python, SQL, XGBoost, Random Forest, Prophet

Delivery: Tableau, D3.js, FastAPI/CLI, AWS
