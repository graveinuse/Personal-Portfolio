Overview

A robust machine learning solution trained on over 11,000 real-world pregnancy records. The system utilizes both lab analytics and clinical scan parameters to accurately predict preterm birth risks, providing medical practitioners with early-warning clinical scoring systems to support preventative care.

Methodology

  • Extracted data from RBC, WBC, TSH, CRL, and Placenta positions.
  • Evaluated baseline models: Logistic Regression, Naive Bayes.
  • Applied advanced ensemble architectures (Random Forest, XGBoost).
  • Achieved a culminating ensemble model with 98.7% accuracy.

Impact & Results

High accuracy predictions identify at-risk patients up to the 35th week of pregnancy, shifting treatment paradigms from reactive to proactive intervention. The solution ultimately supports doctors in reducing infant mortality rates tied to preterm complications.

Technologies

Languages & Frameworks: Python (Flask, Pandas, Scikit-learn), HTML5, CSS3, Bootstrap 4, JS

Environment: Jupyter Notebook, Distributed Web Application Environment
