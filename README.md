## 📖 Overview
This repository contains the source code for my personal portfolio. The design language takes heavy inspiration from structured database environments—utilizing thick borders, monospace typography (`Space Mono` and `Syne`), and tabular grid formats designed optimally for a Data Analyst profile.

It allows recruiters and collaborators to seamlessly explore my clinical and demographic predictive models through a clean, programmatic interface.

---

## ✨ Features
- **📊 Genesis Theme:** A clean, data-heavy "dashboard" UI prioritizing typography and tabular structure over bloated frameworks.
- **🔍 Active Project Filtering:** Native JavaScript-driven live search and department-based filtering capabilities.
- **⚡ Scroll Animations:** Dynamic `IntersectionObserver` logic that triggers smooth sub-component fade-ins precisely as users navigate vertically.
- **📧 FormSubmit AJAX Contact:** A fully integrated "Send Message" client form that completely bypasses rigid page reloads utilizing asynchronous REST POST requests.
- **📂 Deep Dive Sub-pages:** Explicit modular environments securely housing comprehensive methodologies, tech stacks, and dataset logic.

## 🛠️ Usage & Local Deployment

Because this project consists of native static documents, there is no bulky backend build process or heavy dependencies required to run it natively.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/graveinuse/Personal-Portfolio.git
   ```
2. **Navigate into the folder:**
   ```bash
   cd Personal-Portfolio
   ```
3. **Launch the site:**
   Simply open `index.html` in any modern browser. For optimal interaction, run a lightweight local instance:
   ```bash
   # If using Python 3:
   python -m http.server 8000
   ```
   Navigate to `localhost:8000` inside your browser.
